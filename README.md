# Drive Shower

A simple Python desktop app that shows all connected storage devices (HDD, SSD, USB).  
Built with PyQt6 and psutil.  
Includes drive details like mount point, file system, total/used/free space.

## How to Run
1. Install Python 3.10+
2. Run `pip install psutil PyQt6`
3. Run `python storage_manager.py`

You can download the `.exe` from the [Releases](https://github.com/YourUsername/StorageDeviceManager/releases) tab.
